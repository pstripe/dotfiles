# Dotifiles

## Applications
This repository contains _my_ dotfiles for

  * alacritty
  * bpytop
  * neovim
  * tmux
  * zsh

## Installation
To install all dotfiles call
```
make install
```

You can install dotfiles for required application by calling
```
make APP_NAME
```
